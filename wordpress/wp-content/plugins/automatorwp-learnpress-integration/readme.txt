Connect AutomatorWP with LearnPress

== Description ==

[LearnPress](https://wordpress.org/plugins/learnpress/ "LearnPress") is a full-featured WordPress LMS plugin that allows you to easily create and sell courses online.

This plugin automatically connects [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") with LearnPress adding new triggers and actions that you can use to connect with other plugins and automate workflows letting you save time and get focused on your most important work.

= Triggers =

* User attempts any/specific quiz.
* User completes any/specific lesson.
* User completes any/specific course.

= Pro version =

[Pro version](https://automatorwp.com/add-ons/learnpress/ "AutomatorWP - LearnPress") available that let's you expand the integration between AutomatorWP and LearnPress bringing you more ways to automatize your elearning website.

= What is AutomatorWP? =

AutomatorWP is an automator plugin that lets you connect your WordPress plugins together and add automation to unlimited workflows.

You can automate anything including sales, marketing, administrative tasks, learning and any other kind of processes you want letting you save time and get focused on your most important work.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
