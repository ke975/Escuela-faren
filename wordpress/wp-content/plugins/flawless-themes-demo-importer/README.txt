== Description ==

Flawless Themes Demo Importer plugin helps you import demo content for various free themes of [flawlessthemes](http://flawlessthemes.com/) . Flawless Themes are dedicated to creating high quality, easy to use WordPress themes. If you want to import demo content for any of the flawless Themes free version, then this is the plugin for you. Remember this only works with Flawless Themes (themes) . This plugin is based on [One Click Demo Import Plugin](https://wordpress.org/plugins/one-click-demo-import). Install the Flawless Theme Demo Importer Plugin, One Click Demo Import Plugin and you are good to go.

== Installation ==

**From your WordPress dashboard**

1. Visit 'Plugins > Add New',
2. Search for 'Flawless Themes Demo Importer' and install the plugin,
3. It will also need 'One Click Demo Import' plugin so install that too
3. Activate both of the plugin from your Plugins page.

== Frequently Asked Questions ==
